VisiData

v0.97

Rapidly explore columnar data in the terminal

VisiData is free, open source, and can be installed in seconds on Linux or MacOS. Just open your terminal and type:

pip3 install visidata

Efficient

Immediately start browsing large datasets while they load in the background. Every command is only one or two keystrokes.

Universal

Open terabyte .csv files, remote postgres databases, JSON, HDF5 files, and any other source that can be presented as tabular data.

Extensible

Add new data sources, views, commands, and entire terminal applications, with a minimum of code and a maximum of flexibility.
