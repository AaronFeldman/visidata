VisiData

v0.98.1

Rapidly explore columnar data in the terminal

VisiData is free, open source, and can be installed in seconds on any Linux or MacOS system with Python3 installed. Just open your terminal and type:

pip3 install visidata

Efficient

Immediately start browsing large datasets while they load in the background. Every command is only one or two keystrokes.

Universal

Open gigabyte .csv files, remote Postgres databases, JSON endpoints, HDF5 files, and anything that can be viewed as tabular data (which is pretty much everything).

Extensible

Add new commands, views, data sources, and even entire applications, with a minimum of code and a maximum of flexibility.
